* Customer management
* Product management
* Billing system
* GST calculation

* Invoice generation

* Sales dashboard
* Chart.js analytics

* Excel reader using OpenPyXL
* Responsive Bootstrap 5 interface
* MySQL database integration
* Session management
* Project documentation

* Input validation
